# congressional-alpha

![Status: In progress](https://img.shields.io/badge/status-in%20progress-blue)

A fully local data, NLP, and backtesting pipeline for analyzing whether U.S. senator stock disclosures from January 2025 onward contain tradeable alpha.

## Data Notes

Yahoo Finance news coverage is recent and uneven by ticker. To keep trades and news in the same temporal window, the effective research universe is congressional trades disclosed January 2025 onwards.
